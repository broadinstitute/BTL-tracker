  4. trait Container extends AnyRef

    A Container is something that content can be put into.

    A Container is something that content can be put into. There can be initial content, for example a Material can be set as the initial content for a tube. Ultimately the contents of a container consists of it's initial contents as well as any additional contents transferred from other components. Those additional contents are never explicitly set anywhere - they are implicit in the graphs created that show the transfers that take place into a container.

  6. trait Location extends AnyRef

    A location is someplace a component can be stored.

    A location is someplace a component can be stored. The location ID must point to another container - for example tubes or plates can be located in a freezer. Locations can also be recursive, meaning that one component can be located in another which in turn is located in another component. For example wells are located on plates and plates can be located in a freezer.

  11. trait Transferrable extends AnyRef

    Transferrable trait indicates that contents of a component can be transferred to other components of the type listed below as valid. A component's contents are both explicit initial contents as well as contents transferred into the component.
