Freezer

Related Docs: object Freezer | package models

case class Freezer(id: String, description: Option[String], project: Option[String], tags: List[ComponentTag], address: String, temperature: Float) extends Component with Product with Serializable

Freezer - it's a component where other components can be stored (i.e., it can be a location for other components). Note that a freezer has an address, not a location. It can't be stored inside another component).
